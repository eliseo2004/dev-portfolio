# Eliseo Martinez's Portfolio

---

## Overview

This is my personal portfolio website built with **Angular**, **TypeScript**, **SCSS**, and **Angular Router**. The project showcases my skills, experience, projects, and technologies while providing a responsive and modern user experience.

---

## Features

- **Responsive Design:** Optimized for desktop, tablet, and mobile devices.
- **Angular Framework:** Built using Angular with a component-based architecture.
- **TypeScript:** Strongly typed code for better scalability and maintainability.
- **SCSS Styling:** Modular and organized styling using SCSS.
- **Multilingual Support:** Integrated internationalization (i18n) for multiple languages.
- **Interactive UI:** Smooth animations, modern layouts, and responsive navigation.
- **Project Showcase:** Displays my personal and professional projects with detailed information.
- **Contact Section:** Easy access to my professional profiles and contact information.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/eliseo2004/dev-portfolio.git
```

### 2. Navigate to the project

```bash
cd portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
ng serve
```

or

```bash
npm start
```

### 5. Open your browser

```
http://localhost:4200
```
